About the Project

Designed to support maternity and birthing home clinics in managing maternal healthcare records efficiently.

Our goal is to provide a user-friendly and reliable tool that empowers clinics to modernize their workflows while remaining accessible and easy to use.

Read Project Documentation

Automates Record-keeping

Instead of writing patient information manually in logbooks, the software stores data digitally.

Organize Patient Data

Patient records are stored in a structured database rather than scattered notebooks.

Simplifies Routine Computation

The app performs automated computations determining patient's Estimated Due Date (EDD), Gestational Age, Trimesters and Scheduled Prenatal Checkup.

Save Valuable Time

With Bloom Birth, maternal healthcare providers can focus more on patient care and less on paperwork.

Installation & Manual

Guide to install this application and how to use it

How to Download?

Click Home located at top-right of navigation. At Home section click "Download BloomBirth Latest Release (1.0.) for Windows 64-bit" located at bottom-left this will proceed to download the file automatically.

How to Install?

After successfully downloading the app, locate the downloaded file named "BloomBirthSetup.exe" in your computer's download folder. Double-click the file to run the installer and follow the on-screen instructions to complete the installation.

Launch/Open the App and Explore

After installation, open it no login required and familiarize yourself with UI and its content.
